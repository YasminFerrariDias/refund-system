# Refund System

Sistema de solicitação de reembolso desenvolvido com React, TypeScript e Tailwind CSS.

## Tecnologias

- [React](https://react.dev/) — biblioteca para construção de interfaces
- [TypeScript](https://www.typescriptlang.org/) — tipagem estática
- [Vite](https://vitejs.dev/) — bundler e servidor de desenvolvimento
- [Tailwind CSS](https://tailwindcss.com/) — estilização

## Como Executar

```bash
pnpm install
pnpm dev
```

## Sobre

Projeto desenvolvido para praticar React com TypeScript e Tailwind CSS, aplicando componentização e tipagem estática.
